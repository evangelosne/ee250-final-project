Instructions to compile:
1. Create a virtual environment in the react-app directory in the VM
2. Run the command "npm start" to deploy the frontend
3. Create a virtual environment in the backend directory in Raspberry Pi
4. Run the command "flask run" to deploy the backend
5. Navigate to the deployed react app and click the button on the screen
6. Navigate to the terminal in the Raspberry Pi and enter in the inputs that populate

External libraries used: grovepi, flask, react
